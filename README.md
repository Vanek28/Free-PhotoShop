# Free-PhotoShop
Це мій міні-проєкт. Хотів зробити щось просте для редагування фоток, щоб не чекати півгодини, поки відкриється справжній Фотошоп. Написав швидко, працює чітко.

### 🤔 Що тут можна робити:
* **Фільтри:** Блюр, контур, різкість, деталі та ефект "тиснення" (Emboss).
* **Колір:** Кнопка для ЧБ та слайдер, щоб підкрутити контраст.
* **Трансформація:** Поворот картинки на 90 градусів та зміна розміру (ширина/висота).
* **Скинути все:** Якщо наклацав зайвого, кнопка "Скинути" поверне все як було.
* **Збереження:** Кнопка "Зберегти" — і твоє творіння вже в папці.

### 🛠 Як запустити:
Тобі треба Python і дві бібліотеки. Відкрий термінал і пропиши:
pip install customtkinter pillow

Потім просто запускай мій файл:
python main.py

### 🎨 Про дизайн:
Використав бібліотеку CustomTkinter, тому кнопки виглядають по-сучасному, а не як у Windows 95. Весь код на чистому Python + Pillow для обробки пікселів.

### 🚀 Плани на майбутнє:
- Додати обрізку (crop).
- Зробити кнопку "Назад" (Undo), бо зараз скидається тільки до оригіналу.
- Може колись додам стікери.
